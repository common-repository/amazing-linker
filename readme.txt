== Description ==

**Amazing Linker** is a WordPress plugin that gives you all the facilities that you need to place your affiliate link in your content in a different format. You will get all the necessary information like price, review, rating. And the data are always up to date by our intelligent information grabber. Amazing Linker follows all the rules and guidelines of Amazon, so you will never get an unwanted ban from Amazon.

== Installation ==

Before installing the plugin please make sure that

1. Your php version is 5.4 or greater
2. Wordpress version is 4.0 or greater

This section describes how to install the plugin and get it working.

e.g.

1. Upload `amazing-linker.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select 'Amazing Linker' from admin menu and setup the necessary settings
4. use our shortcode
5. Done!
